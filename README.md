ParrotBrowser

ParrotBrowser is a high-performance, minimalist web browser for Windows. It combines the modern Chromium rendering engine with a custom-built interface that allows for total visual personalization.
Getting Started

To run the browser, simply open the ParrotBrowser.exe file located in this folder.
Folder Contents

For the browser to function correctly, the following files must remain in the same folder:

    ParrotBrowser.exe: The main application.

    WebView2Loader.dll: The engine that powers web page loading.

    D3DCOMPILER_47.dll: Ensures the graphics run smoothly on all Windows versions.

    config.ini: Created automatically to store your custom themes and colors.

Features
Intelligent Search

The address bar at the bottom is "smart." You can type a direct website address (like google.com) or simply type a word or phrase to search Google automatically. Press Enter to navigate.
The Taskbar Sidebar

Click the Taskbar button on the top right to open the side menu. From here you can:

    Access History: See a list of the sites you have visited during this session and click them to jump back instantly.

    Open Settings: Access the appearance customization menu.

Appearance Customization

ParrotBrowser allows you to change its look in real-time:

    Accent Colors: Use the color wheel in the Settings menu to change the color of buttons, borders, and highlights.

    Dark/Light Mode: Switch between themes to suit your environment.

    Auto-Save: You don't need to click "Save." Your colors and theme choices are remembered automatically the next time you open the browser.

Requirements

    Windows 10 or 11 (recommended).

    Microsoft Edge WebView2 Runtime: Most modern PCs have this installed. If the browser doesn't load pages, please install the official "Evergreen Bootstrapper" from Microsoft.

Tips

    If the web content is "covering" your menus, ensure you are using the Taskbar button to open the sidebar; the browser is designed to intelligently hide itself when you are adjusting your settings.

    To reset the browser to its original green look, you can simply delete the config.ini file while the browser is closed.
